#用于修改用户资料
chfn 选项 参数 用户名

#常用选项
-f name		#设置用户真是姓名
-h phone	#设置家庭电话
-o address	#设置办公地址
-p phone	#设置办公电话
-u		#显示帮助信息
-v		#显示版本信息

#注意
1.若不指定任何选项,chfn会进入问答式界面.

#示例
chfn youyou		#设置用户信息
chfn -f yang youyou	#设置用户姓名
#通过脚本更改用户资料
