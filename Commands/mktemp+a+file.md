#用于创建临时文件
mktemp 选项 临时文件名模板

#常用选项
-q			#发生错误时不产生信息
-p 路径名		#指定临时文件所在位置
-t			#临时文件存在环境变量指定位置处
-d			#创建临时目录,而不是临时文件

#注意
1.如果直接使用mktemp不加任何选项,会在/tmp目录下产生tmp.XXXXXX的临时文件.
2.模板众必须有XXXXXX,在创建后会将X替换为其他随机字符
3.环境变量中的TMPDIR存有临时盘存储路径,-t选项下默认存在此处

#示例
mktemp			#在/tmp下创建临时文件
mktemp ~/mytemp.XXXXXX	#在~下创建临时文件
